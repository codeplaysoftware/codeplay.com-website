The increasing use of machine learning has been the biggest topic in both software and hardware circles during the past few years, and this will continue as it is deployed to a range of products from smart speakers to self-driving cars.

At Codeplay we have a vision to make it easy for developers to be able to write machine learning applications that can be accelerated on a wide range of processors. This is why we began hosting the Machine Learning Edinburgh meetup group. We want to provide a forum for everyone in Edinburgh and beyond to be able to learn about the latest techniques in this field.

We are also pleased to publish videos of the talks from our last meetup.

Join us at our next Machine Learning Edinburgh meetup on April 6th by registering on the Meetup website.
