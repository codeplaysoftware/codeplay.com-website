The Khronos™ Group have released the provisional specification for version 2.0 of the Standard Portable Intermediate Representation (SPIR™) for OpenCL™.

"SPIR 2.0 will let tools companies like Codeplay produce tools that allow more languages to be accelerated with heterogeneous processors. The new features of SPIR 2.0 will make it possible to provide very easy to use acceleration for a wide range of programming languages," said Andrew Richards,CEO, Codeplay .
