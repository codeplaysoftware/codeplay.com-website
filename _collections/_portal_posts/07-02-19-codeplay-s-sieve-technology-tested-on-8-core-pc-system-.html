Codeplay has succesfully tested programs written using the Sieve Parallel Processing System on a dual Intel® Xeon® 5300 system. The results showed 740% performance increase executing in parallel on 8 cores compared to that on a single core.

The test programs written in Sieve C, which were originally created to test Sieve on a dual-core system, were run on the 8-Core PC with no changes to the code or re-compilation. These tests show the Sieve system to be a powerful and easy to use system for writing scalable and portable C/C++ programs which fully exploit all available processing elements.
