The -snvsips3 command line option is used to switch Offload into the SN Visual Studio Integration for PlayStation 3 mode, which means for a given input C++ file, an object/elf file type is produced. The option will make Offload™ automatically call the produced OutputC compile.bat, negating the need for the user to explicity perform that operation.
