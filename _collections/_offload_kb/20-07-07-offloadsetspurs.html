Include: <liboffload> 

int	offloadSetSpurs( void* spurs );

Provides a SPURS instance for the Offload™ runtime to use.

Content has been removed.

A non-zero return value indicates an error has occured.

If you intend to provide a SPURS instance to the Offload™ runtime it is essential you do so before spawning an Offload™ block. If an Offload block is run before this is called behaviour is undefined.
